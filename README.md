## 🛠 Main Architecture

The toolkit features a streamlined, high-level dashboard organized into four core menu modules:

- **Clinical** – Dedicated medical sub-menu consolidating MELD-Na, MELD 3.0, and eGFR liver health diagnostics.
- **Calculator** – Specialized engines for everyday mathematical and functional computing.
- **Financial** – Advanced growth utilities, including monthly compounding, profit margin analysis, and universal loan solvers.
- **Media** – Dedicated tools and hubs for local processing, video utility pipelines, and graphic workflows.
