1. Overview & Good Faith Statement

The Sol-Lee-Soft Digital Suite is a collection of static web tools designed to provide quick, accessible metrics for clinical, financial, and economic analysis.

"These tools were created in good faith with an intent to be accurate, however bugs can happen and we are not charging for these tools and the results are not guaranteed."

2. Operational Logic

Clinical Metrics Dashboard

  • Methodology: Implements industry-standard clinical formulas (MELD-Na, MELD 3.0, eGFR CKD-EPI 2021).
  • Constraint: Requires complete physiological data inputs to generate a score.

Universal Finance Tool

  • Universal Solving: This tool is designed to solve for any single missing variable. Users must input exactly three values (Principal, Rate, Term, or Payment) and leave the target field blank to calculate the result.
  • Technical Note: For non-linear variables such as Interest Rate, the script employs iterative mathematical approximations to reach a result.

Inflation & Payout Toolkit

  • Historical Data: Utilizes hardcoded Consumer Price Index (CPI) data for the years 1900 through 2025.
  • Projection Phase: Values for the year 2026 and beyond are treated as projections. These calculations use the user-defined annual inflation estimate (defaulting to 3%) to compound value into the future.
